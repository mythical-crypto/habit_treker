## What I do
- Set up scalable i18n architecture for Next.js applications
- Manage translation files and key synchronization
- Implement locale-aware formatting for dates, numbers, currencies
- Provide RTL (Right-to-Left) language support
- Handle complex pluralization rules
- Integrate with next-intl or react-i18next

## When to use me
Use this skill when implementing multi-language support, managing translations, or formatting locale-specific data. Essential for projects targeting international users.

## Key Patterns
- Use hierarchical JSON structures for translations
- Implement locale detection and user preference handling
- Use Intl API for date, number, and currency formatting
- Support RTL languages with CSS logical properties
- Handle complex pluralization with CLDR categories
- Keep translation keys semantic and organized
