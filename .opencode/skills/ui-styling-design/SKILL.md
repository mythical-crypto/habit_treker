## What I do
- Guide shadcn/ui component integration and composition
- Enforce Tailwind CSS utility-first styling patterns
- Implement advanced theming with CSS variables and dark mode
- Ensure accessibility (ARIA, keyboard navigation, focus states)
- Provide responsive and mobile-first layout patterns
- Automate component installation and configuration

## When to use me
Use this skill when building UI components, implementing design systems, configuring themes, or ensuring accessibility. Follows the project's design tokens from PROJECT_OVERVIEW.md.

## Key Patterns
- Use design tokens from PROJECT_OVERVIEW.md (no hardcoded colors)
- No 1px solid borders for sectioning — use tonal layering
- Prefer gap over space-y for layouts
- Use semantic coloring with Tailwind classes
- Ensure ARIA attributes and screen-reader labels
- Use asChild pattern for component composition
